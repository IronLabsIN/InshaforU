## Getting Started

### 1. Install dependencies

```sh
npm install
```

### 2. Run the app locally

```sh
npm start
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Build for production

```sh
npm run build
```

Copy the files from `/build` to your hosting solution (Netlify, Vercel, etc).
